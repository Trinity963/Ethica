Sovereign by default. You own the stack from the first command.

Clone the Forge

The public forge lives on GitHub. Clone it, set up your virtual environment, and you have the full Ethica architecture locally.

git clone github.com/Trinity963/Ethica
STEP 02

Pull EthicaGuide

Pull EthicaGuide-7b via Ollama. Ask it anything about Ethica's architecture, modules, or the surgical patch method.

ollama pull Nine9/ethicaguide

Activate the Sanctuary

python3 main.py
