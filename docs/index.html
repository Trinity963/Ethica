Get Started

Up in minutes.
Sovereign from day one.

Ethica runs locally. Your data never leaves your machine. Requires Python 3.10+, Ollama, and a GPU for best performance.

01

Clone the repository

git clone https://github.com/Trinity963/Ethica

02

Run setup

bash setup.sh — installs dependencies and configures the vault.

03

Pull EthicaGuide

ollama pull ethicaguide — your AI build companion, trained on 80 real sessions.

04

Launch

python3 main.py — the Sanctuary wakes.
