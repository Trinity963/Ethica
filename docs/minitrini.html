Origin

MiniTrini — NOI Pulse is a fully local, multi-agent AI desktop platform built from scratch over twelve sessions. Trini is the AI core — she boots with memory, reads Ethica's heartbeat on startup, and writes her own.

Drop any file into the canvas and she reads it. Build multi-step YAML flows. Swap models mid-session. The guard seals every file with SHA-256. Rollback restores the last clean state in one click.

Capabilities

What Trini can do

Every feature runs locally. Every file stays on your machine. Every model is yours to choose.

◈
Multi-Agent Chat

Full conversational AI with session memory. Multi-agent support with tone awareness. Switch agents mid-session. Trini remembers across restarts via the Living Memory Pipeline.

⬡
Canvas

Drop any file — Trini reads and responds automatically. Supports text, code, PDF, DOCX, images, audio, video, archives. Two drop zones. Auto-trigger on drop.

⟁
YAML Flows

Multi-step AI flows via YAML. Per-node model selection. Canvas auto-trigger. Human-in-the-loop pause node. Orchestrate complex pipelines without writing code.

◎
Tool Suite

Optimize, Refactor, Explain, Summarize, Improve Prompt, Analyze, Expand, Reduce, Web Search, Auto Research, Module Scaffold, Guard Bootstrap — all local.

▣
Integrity Guard

SHA-256 seal over 59 files. Heal and seal from inside the UI. Auto-snapshot before every self-patch. Rollback restores the most recent clean state in one click.

∿
VIVARIUM Bridge

Pings Ethica on boot via HTTP. Reads shared_state.json. Writes her own heartbeat. The VIVARIUM dashboard shows all three bodies in one view — two green, one waiting.
